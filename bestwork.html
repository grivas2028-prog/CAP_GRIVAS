My Best Work

Unit 5 CAP Project

Description: This project is an interactive app I created using App Lab based on the show Stranger Things. Users answer a series of questions, and based on their answers, the app determines which Stranger Things character matches their personality. It includes user input, buttons, and different screens to create an engaging experience. The app demonstrates my ability to use coding logic and design to build something functional.

This is an example of my best work because I made sure to meet all the requirements from the rubric and put in my best effort.

I am proud of the design, including the colors, pictures, and backgrounds that make my project stand out and interesting to interact with.

This project was difficult because I sometimes stuggle with coding,especially when trying to make things interactive.

To complete this, I used skills from previous assignments and applied what I learned to build something better.
