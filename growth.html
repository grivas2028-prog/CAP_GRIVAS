My Growth

My Favorite Things Website

Description: This project is a personal website I created using HTML and CSS where I shared some of my favorite things. It includes images, text, and basic styling to organize the content and make the page visually appealing.

This project shows my growth in making a website using HTML and CSS. It helped me get better at organizing content and styling a page.

At first I had trouble setting up my website and making it look how I wanted. For example, I stuggled with placing images and text in the right spots, but I got better at using divs and CSS to organize everything.

The hardest part for me was getting the layout to look right. I got past this by testing my code often and making small changes until it worked.

If I could improve this project, I would make it more creative and explore more layout ideas.
